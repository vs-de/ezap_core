#Ezap Core

Ezap Core Gem for Ezap distributed service system

## Installation

### Prerequisites

  zeromq, either all v2 or all v3:
  http://zeromq.org/

### Fetch it
Add this line to your application's Gemfile:

    gem 'ezap_core', git: 'https://github.com/vs-de/ezap_core.git'

And then execute:

    $ bundle
